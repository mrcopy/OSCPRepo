Nmap scan types:
-sS SYN scan
-sT TCP full connect
-sU UDP scan
-sI Idle scan (requires zombie)
-sV Version scan
-O OS scan

Other flags
-T[0-5] Scan timings
-g src_port Specify the source port to spoof
--min-rate Specify the minimum rate to send packets
--max-retries Specify the max number of retries to scan a port
--script script Specify nse script


Idle scanning
nmap -O -v ---script ipidseq <target> Attempt to identify IP ID incremental for zombie candidate
nmap -Pn -sI zombie[:src_port] <target> Use idle zombie to scan
