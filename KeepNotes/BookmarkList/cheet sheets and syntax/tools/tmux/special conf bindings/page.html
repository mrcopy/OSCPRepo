Prefix: Ctrl + C New Pane
Prefix: Ctrl + A
Prefix + Ctrl + c:   New session
Prefix + Ctrl + f:   Find session
Prefix + |:         Split vertical
Prefix + -:         Split horizontal
ALT + Arrow Keys:  Navigation
Prefix + hjkl:       Navigation
Prefix + HJKL:     Resize
Prefix + <:         Swap pane previous
Prefix + >:         Swap pane next
Prefix + +"         Maximize current pane to a new window
Prefix + Ctrl + h:   Select previous window
Prefix + Ctrl + l:   Select next window
Prefix + tab:       Select last active window
Prefix + m:         Mouse Mode
Prefix + ENTER:     Enter copy-mode
    Copy Mode:
    v:               Begin selection
    Ctrl + v:       Toggle between blockwise visual mode and visual mode
    y:               Copies selection to top paste-buffer
    ESC:             Cancels current operation
    H:               Jump to start of line
    L:               Jump to end of line
Prefix + b:         List paste buffers
Prefix + p:         Paste from top paste buffer
Prefix + P:         Choose which buffer to paste from

Prefix + s:           "Send pane to" window #
Prefix + j:           "Get pane from" window #

Sessions
tmux list-sessions List all current tmux sessions
tmux kill-session -t # Kill a speicfic session
tmux kill-server Kill all sessions
