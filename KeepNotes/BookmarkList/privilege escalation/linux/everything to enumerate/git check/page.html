Companies may have internal or private git or svn repositories that you can't OSINT ahead of time. If you find a git or svn repository, check the logs for interesting things! Accidental keys/passwords

Found a .git repository, so let's check:
git log

If you find something interesting, you can check with
git diff <commit hash>

Read packed-refs (Git compresses necessary refs into this file).
You can checkout any refs you find
git branch -r

Check any interesting tags (and display them with: git show <tag>)
git tag
