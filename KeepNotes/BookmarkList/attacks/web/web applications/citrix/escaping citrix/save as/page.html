One of the most common escape methods is to invoke "Save As" dialogue box.
You can access this any number of ways. Many executables, web browsers (view source if you need to), etc.

Once in the "Save As" box, you can try a few things:
1: in the address bar, type 'cmd.exe' or 'powershell.exe'
2: in the address bar, go to cmd.exe or powershell.exe and attempt to run
3: save a .bat file and execute it
