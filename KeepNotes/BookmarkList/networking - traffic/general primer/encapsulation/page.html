Protocols work together through encapsulation. The entire upper protocol packet (header + payload) IS the payload of the lower level protocol.

Examples for TCP/IP four layers:
Application
Transport
Network
Data Link

Every protocol adds its own header to the packet, treating it as a payload.
This happens to every packet sent by a host.

H = header
P = Payload

Application   H[P] -> H[P]
Transport     H[H[P]] -> H[P]
Network  H[H[H[P]]] -> H[P]
Data Link H[H[H[H[P]]]] -> H[P]

The receiving host does the same operation in reverse order.
