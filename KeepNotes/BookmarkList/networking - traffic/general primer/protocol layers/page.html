1984 the International Organization for Standardization (ISO) published a theoretical model for network system communication: the Open System Interconnection model (OSI).

The ISO/OSI model has never been implemented, but it is widely used in literature or when talking about IT networks.

ISO/OSI consists of seven layers from top to bottom:
7 - Application
6 - Presentation
5 - Session
4 - Transport
3 - Network
2 - Data Link
1 - Physical
