Script used to find a dns zone transfer without knowing the domain name of a local machine
10.10.10.83 is the IP of the machine with DNS open (ie nameserver)
dnsrecon will query all local addresses and add the results to olympus.db

dnsrecon -n 10.10.10.83 -r 10.0.0.0/8 -db olympus.db
dnsrecon -n 10.10.10.83 -r 172.16.0.0/12 -db olympus.db
dnsrecon -n 10.10.10.83 -r 192.168.0.0/16 -db olympus.db
