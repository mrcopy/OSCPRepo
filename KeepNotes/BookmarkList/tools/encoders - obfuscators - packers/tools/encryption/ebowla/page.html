Download GO 1.6 
Extract to anywhere (recommended /usr/lib)
Add /usr/lib/go/bin to PATH or link/put binaries in path
Edit genetic.conf
  Tips: Make a generic binary by using hostname or other shared variables. If you encrypt based on username (for example) you won't be able to exec a system shell (because it's tied to the encryption value of the user)
python ebowla <target_binary> genetic.conf
compile as needed (go is ./build_x86_go.sh output/binary.go <output_exe>
