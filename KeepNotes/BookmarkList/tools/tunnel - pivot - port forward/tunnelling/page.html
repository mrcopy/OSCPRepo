DNS
  • iodine
    • This is a piece of software that lets you tunnel IPv4 data through a DNS server. This can be usable in different situations where internet access is firewalled, but DNS queries are allowed.
  • dnscat2
    • Welcome to dnscat2, a DNS tunnel that WON'T make you sick and kill you! This tool is designed to create a command-and-control (C&C) channel over the DNS protocol, which is an effective tunnel out of almost every network.
  • ThunderDNS 
This tool can forward TCP traffic over DNS protocol. Non-compile clients + socks5 support.



ICMP
  • fraud-bridge
    • fraud-bridge allows to tunnel TCP connections through ICMP, ICMPv6, DNS via UDP or DNS via UDP6. Project, not stable
  • tcpovericmp
    • TCP implementation over ICMP protocol to bypass firewalls
  • icmptunnel
    • Transparently tunnel your IP traffic through ICMP echo and reply packets.


SMB
  • Invoke-Pipe
    • Forward local or remote tcp ports through SMB pipes


TCP
  • Shootback
    • Python Reverse TCP tunnel let you access target behind NAT or firewall
  • SSF
    • Secure Socket Funneling TCP and UDP port forward, SOCKs, remote shell, cross platform
  • PowerCat
    • PowerShell TCP/IP swiss army knife


UDP
  • udp2raw-tunnel
    • UDP traffic into Encrypted/fakeTCP/ICMP traffic using Raw Sockets (UDP Firewall bypass)


Web / HTTP
  • Multitun
    • Efficiently and securely tunnel everything over a harmless looking WebSocket!
  • Stunnel
    • Stunnel TLS Proxy
  • SharpSocks
    • Tunnellable HTTP/HTTPS socks4a proxy written in C# and deployable via PowerShell
  • reGeorg
    • Successor to reDuh, create SOCKs through the DMZ
  • RedSocks
    • redirect TCP connection to SOCKs or HTTPS proxy using your firewall, TCP-to-proxy redirector
  • Tunna
    • Wrap and tunnel any TCP communication over HTTP
  • httptunnel
    • Tunnel through encapsulated HTTP traffic
